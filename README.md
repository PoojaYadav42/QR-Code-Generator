# QR-Code-Generator
A lightweight and responsive web application that generates QR codes from any text or URL using HTML, CSS, and JavaScript. The app works completely on the client side and does not require any backend.

# Features:

 Generate QR codes for any text or URL

 Adjustable QR size (128px, 256px, 512px)

 Select error correction levels (L, M, Q, H)

 Real-time preview of generated QR code

 Download QR as PNG

 Copy input text/URL to clipboard

 Fully responsive UI

 No server required (100% client-side)

# Technologies Used:

HTML5 – Structure and layout

CSS3 – Styling and responsive design

JavaScript (ES6) – Core functionality

QRCode.js – JavaScript library for QR code generation
